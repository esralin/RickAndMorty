# Rick and Morty Swift App
A Rick and Morty application to list characters from [The Rick and Morty API.](https://rickandmortyapi.com/)

 - Used UIKit
 - MVVM Architecture
 - Implemented by using JSON Api Service
 - Based on Auto layout
